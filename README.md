# Barcode - a year of FT

A year of FT News presented as a barcode

![Alt text](./docs/example.png?raw=true "Example of a year of FT headline images in Barcode format")

## Setup

```
cp .env_example .env
npm install
```

## Running

```
nodemon index.js
```

Go to [http://localhost:3000](http://localhost:3000)
